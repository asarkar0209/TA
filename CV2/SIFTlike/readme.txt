This code is implements and demonstrates the texture recognition published in the following paper:

Yong Xu, Sibin Huang, Hui Ji, Cornelia Fermuller,Scale-space texture description on SIFT-like textons, CVIU, 2012

The codes in 2D directory are used for static texture and that in 3D directory are used for dynamic texture. There are 2 demos provided in each set of code.

Yong Xu, Sibin Huang, Hui Ji, Cornelia Fermuller, May, 2012.
